# MyCalendar üìÖ

MyCalendar is a modern, feature-rich calendar application built with **Vue 3** and **Tailwind CSS**. It is designed to support multiple calendar systems, making it perfect for users who need to switch between **Gregorian**, **Jalali (Persian)**, and **Hijri** dates seamlessly.

![MyCalendar Banner](https://placehold.co/1200x400/1e293b/ffffff?text=MyCalendar+App)

## ‚ú® Features

- **Multi-Calendar Support**: View and navigate dates in Gregorian, Jalali, and Hijri systems.
- **Smart Reminders**:
  - Set reminders for any specific date and time.
  - **Persistent Storage**: Reminders are saved automatically and persist across sessions.
  - **Notifications**: Receive beautiful in-app toast notifications and browser alerts with sound.
  - **Editing**: Easily edit or delete existing reminders.
- **Date Converter**: A dedicated tool to convert dates between Jalali, Gregorian, and Hijri formats instantly.
- **Modern UI/UX**:
  - **Dark Mode**: Fully supported dark theme for comfortable night usage.
  - **Responsive Design**: Works perfectly on desktop and mobile devices.
  - **Animations**: Smooth transitions and micro-interactions powered by custom CSS animations.
- **Year Selection**: An intuitive, full-screen grid interface for quick year navigation.

## üöÄ Technologies Used

- **[Vue.js 3](https://vuejs.org/)**: The progressive JavaScript framework.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
- **[Moment.js](https://momentjs.com/)**:
  - `moment-jalaali`: For Persian date manipulations.
  - `moment-hijri`: For Islamic date manipulations.
- **Vite**: Next Generation Frontend Tooling.
